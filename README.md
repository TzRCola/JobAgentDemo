# 🤖 多Agent协作求职助手

一个基于大语言模型的多Agent协作系统，帮你自动化求职全流程：市场分析 → 简历撰写 → 面试准备。

## ✨ 功能

- **📊 市场分析 Agent**：搜索目标岗位的真实招聘信息，分析JD，提炼核心技能
- **✍️ 简历撰写 Agent**：基于真实背景，三级包装策略，生成有竞争力的简历
- **🎤 面试准备 Agent**：严苛面试官视角，生成追问清单和防守策略
- **🧠 持续学习**：从反馈中自动提炼偏好规则，越用越懂你
- **👥 多用户支持**：独立档案，互不干扰
- **📥 一键导出**：Markdown格式完整求职报告

## 🚀 快速开始

```bash
pip install -r requirements.txt
python agent_v10.1.py
需要配置智谱AI和SerpAPI的API密钥。
```
## 🏗️ 架构
```
用户输入 → 主控Agent（意图识别 + 调度）
              │
    ┌─────────┼─────────┐
    ▼         ▼         ▼
📊 市场分析  ✍️ 简历撰写  🎤 面试准备
```
